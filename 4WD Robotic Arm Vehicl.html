4WD Robotic Arm Vehicle

A 4-wheel drive mobile robot with a 5-DOF arm for automated dustbin pickup and disposal. Controlled via Bluetooth or automatic sequence.

Project Overview

This project integrates a 4WD chassis with a 5-DOF robotic arm. The base uses 4 DC geared motors driven by an L298N driver, and the arm uses servo motors (controlled via PCA9685 or directly). Control modes include Manual (via Bluetooth app) and Automatic sequence (pickup ‚Üí move ‚Üí drop ‚Üí return).

Key Features

  • 4-wheel drive chassis with differential control
  • 5-DOF robotic arm for gripping & lifting small dustbins
  • Bluetooth control via MIT App Inventor / Custom mobile app
  • Automatic pickup and disposal routine
  • Obstacle sensing (optional: ultrasonic / IR sensors)

Sample Code

// === Arduino Uno + L298N (simplified) ===
const int ENA = 9, IN1 = 8, IN2 = 7;
const int ENB = 10, IN3 = 6, IN4 = 5;

void setup() {
  pinMode(ENA, OUTPUT); pinMode(IN1, OUTPUT); pinMode(IN2, OUTPUT);
  pinMode(ENB, OUTPUT); pinMode(IN3, OUTPUT); pinMode(IN4, OUTPUT);
}

void forwardMove(int speed=200){
  digitalWrite(IN1, HIGH); digitalWrite(IN2, LOW);
  analogWrite(ENA, speed);
  digitalWrite(IN3, HIGH); digitalWrite(IN4, LOW);
  analogWrite(ENB, speed);
}

void stopAll(){
  analogWrite(ENA, 0); analogWrite(ENB, 0);
}

void loop(){
  forwardMove(200); delay(1200);
  stopAll(); delay(400);
}
